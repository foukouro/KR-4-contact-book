# Книга контактов

React-приложение для управления контактами

## Что сделано

Добавление контактов (имя и телефон)
Просмотр списка контактов  
Удаление контактов
Сохранение в LocalStorage
Валидация полей

## Технологии

- React
- CSS
- LocalStorage

## Как запустить

```bash
npm start
```
Приложение откроется в браузере по адресу: http://localhost:3000
